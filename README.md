# Realtime Random Chat: Server Side

- Go
- gorilla websocket
- glide

# Running Server

```shell
$ go build

$ ./go-websocket-chat
```

# Running Client

```shell
$ cd client/w-sock
$ npm install
$ npm start
```

Or with specific `PORT`

```shell
$ PORT=3000 npm start
```

Visit http://localhost:3000
