# Realtime Random Chat

## Server Side
Server side uses Go and Gorrilla

## Client Side
Client side uses NodeJs and Express

# Running Server

```shell
$ go build

$ ./go-websocket-chat
```

# Running Client

```shell
$ cd client/w-sock
$ npm install
$ npm start
```

Or with specific `PORT`

```shell
$ PORT=3000 npm start
```

Visit http://localhost:3000
